# Contributing to Speech-to-Text App\n\nWe love your input! We want to make contributing to this project as easy and transparent as possible.\n\n## Development Setup\n\n```bash\n# Clone the repository\ngit clone https://github.com/yourusername/speech-to-text-app.git\ncd speech-to-text-app\n\n# Install dependencies\nnpm install\n\n# Create .env file with your Deepgram API key\necho \"DEEPGRAM_API_KEY=your_key_here\" > .env\n```\n\n## Running Locally\n\n**Terminal 1:**\n```bash\nnode server.js\n```\n\n**Terminal 2:**\n```bash\nnpm run dev\n```\n\nOpen `http://localhost:5173` in your browser.\n\n## Code Style\n\n- ESLint is configured; run `npm run lint` before committing\n- Use 2-space indentation\n- Follow existing code patterns\n\n## Pull Request Process\n\n1. Fork the repository\n2. Create a feature branch: `git checkout -b feature/your-feature`\n3. Make your changes\n4. Run `npm run lint` to check code style\n5. Commit: `git commit -m \"Add your feature\"`\n6. Push: `git push origin feature/your-feature`\n7. Open a Pull Request\n\n## Issues\n\nBefore opening a new issue, check existing ones. When submitting an issue, include:\n- Clear description\n- Steps to reproduce (bugs)\n- Expected behavior\n- Your OS and Node version\n\n## Questions?\n\nFeel free to open a GitHub Discussion or Issue.\n\nThank you for contributing! 🎉\n"
